Mate del loco

El proyecto consiste en crear un tablero de ajedrez con piezas minimalistas. 
Las piezas se moverán para representar el jaque mate del loco el cual consiste en 4 jugadas. 
Al finalizar, las piezas volverán a su posición inicial. 

Características 

	* Habrá una textura en el tablero para definir los escaques.
	* El reloj que marca el tiempo de juego también tendrá una textura.
	* Habrá una luz que genere sombras en las piezas.
	* Animación del mate del loco.

¿Como correrlo?

Dentro de la carpeta del proyecto, abre una terminal y corre un Servidor Simple de python. 

Python 2.x

python -m SimpleHTTPServer

Python 3

python -m http.server

Despues en tu navegador ve a localhost:8000
